🎯 PROJECT STRUCTURE PATTERN
───────────────────────────────────────────────────────────────────────

Pattern: Repository → Service → Controller → Route

Example Flow:
1. UserRepository
   └─ Database queries (CRUD)

2. AuthService
   └─ Business logic (hash password, generate token)

3. AuthController
   └─ Handle HTTP requests

4. AuthRoute
   └─ Define endpoints (POST /login, POST /register)

This pattern is repeated for:
• User/Auth
• Courses
• Enrollments
• Progress
• Payments
• Reviews
• Tests
• etc.
